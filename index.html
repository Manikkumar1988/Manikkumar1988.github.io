-get-started-

Follows 9 simple steps to build app and publish to google playstore. Keep scrolling to see em'.
The steps mentioned here have been practised over the years and gets adapted as needed. If you think you are pro, then scroll faster. xP

One
Step - 1 - Gathers Requirement.
Conceptualize the app from Wireframe, Competitor app, requirement specification document (or even a pencil sketch)
Two
Step - 2 - Tracks Requirement.
Jot down the features tagging with expectant release version using JIRA, Assembla, Trello (or an excel!)
Three
Step - 3 - Writes Test Case.
Ensure quality abiding xtreme programming methodology, develops test case Espresso, JUnit
This is the place where deliverable of quality (or bug-free) app is ensured even before the actual implementation started.
Four
Step - 4 - Implements Functionality
Implements feature using Android Studio backed by Gradle
Five
Step - 5 - Builds Daily.
Ships build daily along with artifacts like test report, environment builds, screen captures or video recording of app
Six
Step - 6 - Automation.
Automates the generation 'Step - 5' artificats using Continous Integration tools like jenkins, Circle
Seven
Step - 7 - Code Storage.
Hosts the code in repository using tools like Git, SVN, CVS or dropbox
Eight
Step - 8 - Publish to Playstore.
Releases to Google playstore
